# 圣遗物预测器 - 轻量版说明

## 简介

这是原神圣遗物词条预测工具的轻量版本，只保留深度学习预测功能，删除了统计预测和模型训练功能，使用更加简单直观。

## 与完整版的区别

| 功能 | 轻量版 | 完整版 |
|------|--------|--------|
| 深度学习预测 | ✅ 保留 | ✅ 保留 |
| 统计预测 | ❌ 删除 | ✅ 保留 |
| 模式切换 | ❌ 删除 | ✅ 保留 |
| 模型训练 | ❌ 删除 | ✅ 保留 |
| 配置编辑器 | ❌ 删除 | ✅ 保留 |
| 词条录入 | ✅ 保留 | ✅ 保留 |
| 胚子功能 | ✅ 保留 | ✅ 保留 |
| 数据保存/导出 | ✅ 保留 | ✅ 保留 |

## 使用方法

### 1. 直接运行

```bash
python artifact_predictor.py
```

### 2. 录入词条

- 点击"五星"标签页录入五星词条
- 点击"三星"标签页录入三星词条（垫刀）
- 录入3个词条后自动显示预测结果

### 3. 查看预测

- **表1**：显示10种词条类型的预测概率
- **概率分布**：基于深度学习模型的智能预测

**预测原理**：
- 3词条：使用 skip=1
- 4词条：使用 skip=1,2 加权
- 5词条：使用 skip=1,2,3 加权
- 6+词条：使用 skip=1,2,3,4 加权

## 打包说明

使用 PyInstaller 打包：

```bash
pyinstaller build_轻量版.spec
```

## 注意事项

1. 需要预训练模型 `强化记录/models/best_model.pt`
2. 首次使用需要录入至少3条历史记录
3. 数据自动保存在 `强化记录/records/` 目录

## 技术特点

- **深度学习**：使用 LSTM 神经网络预测词条概率
- **轻量快速**：删除训练功能，启动更快
- **简单易用**：无需配置，开箱即用
