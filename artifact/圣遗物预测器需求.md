# 原神圣遗物预测器需求

## 一、核心功能需求

### 1. 界面布局
- **左侧多标签页设计**：包含五星、三星、胚子三个标签页
  - 五星标签页：录入五星词条（显示具体数值如f16, f19等）
  - 三星标签页：录入三星词条（显示具体数值如f8, f9, f10, f11等）
  - 胚子标签页：录入强化目标胚子（一般有四个副词条）
- **中间录入区**：共用的录入区，可滚动显示无限个槽位
  - **最新词条在最上面**：新录入的词条显示在列表顶部
  - 显示内容：档位、词条（具体数值）、录入顺序
- **右侧预测结果区**：共用的预测界面
  - 表1：预测下一次词条（显示概率%）
  - 预测模式选择：统计模式/深度学习模式
  - 操作按钮：保存数据、导出Excel、训练模型
  - 状态显示：记录数、模型状态

### 2. 词条输入
- **五星词条**：显示具体数值（f16, f19, f21, f23等），4个档位
- **三星词条**：显示具体数值（f8, f9, f10, f11; s100, s115, s129, s143等），4个档位
- **混合预测**：三星和五星的前三个词条共同预测结果
- **数据隔离**：左侧输入按钮的状态不共享，但中间录入区共用显示

### 3. 录入区功能
- **无限槽位**：可以滚动显示无限个录入槽位
- **最新词条置顶**：新录入的词条显示在最上面（第1条是最新的）
- **显示内容**：档位、词条（具体数值）、录入顺序（从1开始递增）
- **删除功能**：可以删除最后一个录入的词条
- **清空功能**：清空所有录入记录

### 4. 数据功能
- **保存数据**：将当前4个词条保存为JSON（需4个词条）
  - 文件：`强化记录/record_YYYYMMDD_HHMMSS.json`
  - 内容：时间戳、4个词条详情（值、档位、星级、类型、顺序）
- **导出Excel**：将当前录入数据导出为Excel
  - 文件：`强化记录/exports/current_entries_YYYYMMDD_HHMMSS.xlsx`
  - 内容：录入顺序、词条值、档位、星级、类型
- **训练模型**：训练深度学习模型（需≥10条记录）

### 5. 预测功能
- **实时预测**：每输入一个词条就实时更新预测结果
- **预测逻辑**：基于过去3个词条（含档位信息）预测第4个词条
- **双模式预测**：
  - 统计模式：使用genmat401.npy查表预测
  - 深度学习模式：使用best_model.pt模型预测（需训练）
- **预测结果显示**：右侧表格显示概率%

---

## 二、使用流程

1. **选择标签页**：在左侧选择五星、三星或胚子标签页
2. **录入词条**：点击词条按钮录入（显示具体数值）
3. **查看录入**：中间录入区显示所有录入的词条（最新在最上面）
4. **查看预测**：右侧实时显示预测结果（概率%）
5. **保存数据**：强化完成后点击"保存数据"
6. **训练模型**：积累10条后点击"训练模型"
7. **切换模式**：训练完成后可切换到"深度学习模式"

## 四、预测算法说明

### 1. 统计模式

**模型数据结构（genmat401.npy）**：
- **S1/S2/S3**：三星词条统计矩阵
- **W1/W2/W3**：五星词条统计矩阵
- **Pcount**: 概率计数基准值

**词条索引映射**：
- **词条类型**：10种（f小防, s小生, g小攻, F大防, S大生, G大攻, j精通, c充能, b暴击, B暴伤）
- **档位数量**：每种类型4个档位
- **总索引数**：40 = 10种类型 × 4个档位
- **索引计算**：`索引 = 类型索引 × 4 + 档位索引`

**预测流程**：
1. 获取最后3个词条
2. 词条转索引
3. 选择统计矩阵（≥2个五星用W3，否则用S3）
4. 查询统计得到10种词条出现次数
5. 计算概率：`概率 = counts / sum(counts) × 100%`

### 2. 深度学习模式

**词条编码**：
```
组合ID = 类型×4 + 档位 + 星级×40

例: f16(5星,档位1) = 0×4 + 0 + 1×40 = 40
```

**模型结构**：
```
输入(3个ID) → Embedding(16维) → LSTM(64×2层) → 全连接 → 输出(10类概率)
```

**训练条件**：≥10条记录

## 五、常见问题

**Q: 为什么深度学习模式是灰色的？**  
A: 需要模型文件存在（best_model.pt）

**Q: 保存会覆盖之前的数据吗？**  
A: 不会，每次保存生成新文件（带时间戳）

**Q: PT模型和NPY文件有什么区别？**  
A: NPY是通用统计模型，PT是个人数据训练的深度学习模型

**Q: 训练需要多久？**
A: 10条约10-20秒，100条约1-2分钟，500条约3-5分钟
