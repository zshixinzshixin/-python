## 1. 启动程序

双击 `圣遗物预测器.exe`

---

## 2. 录入词条

1. 点击界面上的词条按钮（大攻击、暴击率等）
2. 词条显示在左侧表格
3. 右侧自动显示预测概率

---

## 3. 保存数据

点击【保存数据】按钮

数据保存到 `强化记录/` 文件夹

---

## 4. 训练模型（重要）

**条件**：至少保存10条记录

**注意**：记录保存在 `强化记录/` 文件夹

**操作**：
1. 点击【训练模型】按钮
2. 等待训练完成
3. 切换到"深度学习模式"

**注意**：
1. 模型保存在 `强化记录/models/` 文件夹
2. 会自动保存最佳模型 `best_model.pt`
3. 每次训练还会保存带时间戳的历史版本（如 `model_20260221_153000.pt`）
4. 可删除 `强化记录/models/` 文件夹，重新训练

---

## 5. 查看预测

- **Top-3推荐**：概率最高的3个词条
- **Top-3统计**（仅深度学习模式）：黑色显示，用于对比统计模式结果
- **置信度**：高(绿色) / 中(橙色) / 低(红色)
- **概率表**：所有词条的概率分布

**💡 智能预测**：录入词条越多，预测越准确（会自动结合近期和中期数据）

**💡 双模式对比**：深度学习模式下会同时显示两种模式的Top-3推荐，方便对比参考

---

## 6. 胚子强化建议

**功能**：评估4词条胚子的强化价值

**操作**：
1. 切换到【胚子】标签页
2. 点击选择4个词条（如：暴击、暴伤、精通、充能）
3. 点击【确认胚子】

**查看结果**：
- **表2**：显示4个词条的条件概率（已归一化为100%）
- **强化建议**：
  - 双暴%：暴击+暴伤的概率之和
  - 歪率：100% - 双暴%（非暴击暴伤词条总概率）
  - 建议：≥60%绿色（建议强化）、50-60%橙色（可考虑）、<50%红色（继续垫刀）
- **风险度**：
  - 计算公式：`max(非bB概率) / min(bB概率)`
  - <1.0x 绿色（安全）：bB概率高于非bB
  - 1.0-1.5x 橙色（警告）：非bB概率略高于bB
  - >1.5x 红色（危险）：非bB概率远高于bB

**示例**：
```
胚子：暴击(30%) + 暴伤(30%) + 精通(35%) + 充能(5%)
双暴%：60% | 歪率：40% | 建议：绿色（建议强化！）
风险度：1.17x (精通35% > 暴伤30%) | 橙色（略危险）
```

## 8. 文件位置

```
圣遗物预测器/
├── 圣遗物预测器.exe   # 主程序（双击运行）
├── 强化记录/          # 数据文件（运行后自动生成）
│   ├── exports/       # Excel导出文件
│   └── models/        # 模型文件
│       └── best_model.pt
└── _internal/         # 程序依赖（无需操作）
    └── genmat401.npy  # 统计模型
```

**注意**：
- 只需关注 `强化记录/` 文件夹，你的数据都保存在这里
- `_internal/` 是程序内部文件，不需要操作
