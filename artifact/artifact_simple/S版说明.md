# 圣遗物预测器 - S版说明

这是原神圣遗物词条预测工具的S版本，只保留深度学习预测功能，删除了统计预测和模型训练功能，使用更加简单直观。

## 使用方法

### 1. 直接运行

```bash
python artifact_predictor.py
```

### 2. 录入词条

- 点击"五星"标签页录入五星词条
- 点击"三星"标签页录入三星词条（垫刀）
- 录入3个词条后自动显示预测结果

### 3. 查看预测

- **表1**：显示10种词条类型的预测概率
- **Top-3推荐**：显示概率最高的3个词条
- **置信度**：显示预测的可靠程度

### 4. 胚子功能

- 点击"胚子"标签页
- 选择4个初始词条
- 查看每个词条的强化概率和建议

### 5. 保存和导出

- **保存数据**：将当前录入的词条保存为JSON
- **导出Excel**：导出为Excel文件

## 预测原理

### Skip策略

根据已录入的词条数量，自动选择不同的skip策略：

| 词条数 | 使用Skip | 说明 |
|--------|----------|------|
| 3 | 1 | 只预测下一个 |
| 4 | 1, 2 | 近期+中期 |
| 5 | 1, 2, 3 | 近期+中期+长期 |
| 6+ | 1, 2, 3, 4 | 四者综合 |

### 权重配置

```python
SKIP_1_WEIGHT = 0.55  # 近期预测权重
SKIP_2_WEIGHT = 0.25  # 中期预测权重
SKIP_3_WEIGHT = 0.15  # 长期预测权重
SKIP_4_WEIGHT = 0.05  # 超长期预测权重
```

## 配置说明

编辑 `config.py` 可调整以下参数：

### Skip预测配置

- `MAX_SKIP`：最大跳过步数（1-4）
- `SKIP_1_WEIGHT`：Skip=1的权重
- `SKIP_2_WEIGHT`：Skip=2的权重
- `SKIP_3_WEIGHT`：Skip=3的权重
- `SKIP_4_WEIGHT`：Skip=4的权重

### 模型结构配置（不建议修改）

- `EMBED_DIM`：词嵌入维度
- `HIDDEN_DIM`：LSTM隐藏层维度
- `NUM_LAYERS`：LSTM层数
- `DROPOUT`：Dropout比率
- `NUM_CLASSES`：输出类别数

## 注意事项

1. **预训练模型**：程序包含预训练模型 `best_model.pt`，无需训练即可使用
2. **词条数量**：至少需要3个词条才能进行预测
3. **数据保存**：建议定期保存数据，避免丢失
4. **模型更新**：如需更新模型，需要替换 `best_model.pt` 文件

## 常见问题

### Q: 为什么删除统计模式？
A: S版专注于深度学习预测，提供更准确的预测结果。

### Q: 如何更新模型？
A: 替换 `强化记录/models/best_model.pt` 文件即可。

### Q: 可以添加训练功能吗？
A: 如需训练功能，请使用完整版程序。

### Q: 预测准确度如何？
A: 基于深度学习模型，准确度取决于训练数据质量。预训练模型已包含大量数据。

### Q: S版和原版预测结果一样吗？
A: **核心预测逻辑完全一致**，S版只是移除了非核心功能，预测结果与原版相同。
