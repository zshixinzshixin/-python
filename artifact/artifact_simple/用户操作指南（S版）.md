# 圣遗物预测器（S版）- 使用指南

## 版本信息
- **版本**：v1.0
- **更新日期**：2026-03-28
- **特点**：深度学习预测，无需训练，开箱即用

---

## 1. 启动程序

双击 `artifact_predictor.exe` 或运行 `python artifact_predictor.py`

**注意**：程序需要 `强化记录/models/best_model.pt` 模型文件才能正常运行

---

## 2. 录入词条

1. 点击界面上的词条按钮（如：大攻击、暴击率等）
2. 词条显示在左侧表格（最新的在最上面）
3. 录入3个词条后，右侧自动显示预测概率

**💡 提示**：录入越多词条，预测越准确（自动使用skip策略）

---

## 3. 查看预测

- **表1**：显示10种词条类型的预测概率
- **概率分布**：基于深度学习模型的智能预测

**预测原理**：
- 3词条：使用 skip=1
- 4词条：使用 skip=1,2 加权
- 5词条：使用 skip=1,2,3 加权
- 6+词条：使用 skip=1,2,3,4 加权

---

## 4. 胚子强化建议

**功能**：评估4词条胚子的强化价值

**操作**：
1. 切换到【胚子】标签页
2. 点击选择4个初始词条类型（如：暴击、暴伤、精通、充能）
3. 点击【确认胚子】

**查看结果**：
- **表2**：显示4个词条的条件概率（已归一化为100%）
- **强化建议**：
  - 双暴%：暴击+暴伤的概率之和
  - 歪率：100% - 双暴%
  - 建议：≥60%绿色（建议强化）、50-60%橙色（可考虑）、<50%红色（继续垫刀）
- **风险度**：
  - 计算公式：`max(非bB概率) / min(bB概率)`
  - <1.0x 绿色（安全）
  - 1.0-1.5x 橙色（警告）
  - >1.5x 红色（危险）

**示例**：
```
胚子：暴击(30%) + 暴伤(30%) + 精通(35%) + 充能(5%)
双暴%：60% | 歪率：40% | 建议：建议强化！
风险度：1.17x (精通35% > 暴伤30%)
```

## 8. 配置调整

如需调整预测参数，编辑 `config.py`：

```python
# Skip预测权重
SKIP_1_WEIGHT_7PLUS = 0.55  # 7+词条时skip=1的权重
SKIP_1_WEIGHT_5TO6 = 0.55   # 5-6词条时skip=1的权重
SKIP_2_WEIGHT = 0.25        # skip=2的权重
SKIP_3_WEIGHT = 0.15        # skip=3的权重
SKIP_4_WEIGHT = 0.05        # skip=4的权重

# 最大跳过步数
MAX_SKIP = 4
```

## 9. 常见问题

### Q: 为什么启动后预测功能不可用？
A: 请检查 `强化记录/models/best_model.pt` 模型文件是否存在。

### Q: 如何更新模型？
A: 替换 `强化记录/models/best_model.pt` 文件即可。

### Q: 可以训练自己的模型吗？
A: 本程序使用预训练模型，不支持自定义训练。如需训练功能，需要专业的深度学习环境。

### Q: 如何打包为EXE？
A:
```bash
pip install pyinstaller
pyinstaller -F -w -i icon/icon.ico artifact_predictor.py
```
**注意**：打包时需要包含 `强化记录/models/best_model.pt` 模型文件。
